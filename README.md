# Notes App

```
go get -u github.com/beego/beego/v2
go get -u github.com/beego/bee/v2
go get -u github.com/beego/beego/orm
bee new beego_notes
```

# Migrations
```
bee generate migration create_notes_table
```

# To Run

```
bee run
```
